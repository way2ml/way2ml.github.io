# 什么是熵，交叉熵, KL Divergence, JS Divergence, f Divergence?

# 熵

熵是信息的度量:

H(p)=−∑ip(xi)logbp(xi)H(p) = -\sum_i p(x_i) log_b p(x_i) H(p)=−​i​∑​​p(x​i​​)log​b​​p(x​i​​)

# 交叉熵

交叉熵:

H(p,q)=−∑ip(xi)logbq(xi)H(p,q) = -\sum_i p(x_i) log_b q(x_i) H(p,q)=−​i​∑​​p(x​i​​)log​b​​q(x​i​​)

其中ppp是真实的概率分布,qqq是预测的概率分布。 通常情况H(p,q)>H(p)H(p,q) > H(p)H(p,q)>H(p)

# KL Divergence

KL Divergence就是:

DKL(P∣∣Q)=H(p,q)−H(p)=−∑ip(xi)logbq(xi)p(xi)D_{KL}(P||Q) = H(p,q) - H(p) = -\sum_i p(x_i) log_b \frac{q(x_i)}{p(x_i)} D​KL​​(P∣∣Q)=H(p,q)−H(p)=−​i​∑​​p(x​i​​)log​b​​​p(x​i​​)​​q(x​i​​)​​

参考: Hands-On Machine Learning with Scikit-Learn & Tensorflow的作者讲解熵,交叉熵,KL散度 (opens new window)

# JS Divergence

JS Divergence是KL Divergence的一种变形:

DJS(p∥q)=12DKL(p∥p+q2)+12DKL(q∥p+q2)D_{JS}(p \| q)=\frac{1}{2} D_{KL}\left(p \| \frac{p+q}{2}\right)+\frac{1}{2} D_{KL}\left(q \| \frac{p+q}{2}\right) D​JS​​(p∥q)=​2​​1​​D​KL​​(p∥​2​​p+q​​)+​2​​1​​D​KL​​(q∥​2​​p+q​​)

  • 值域范围 JS散度的值域范围是[0,1]，相同为0，相反则为1。相比较于KL，对相似度的判别更准确了。
  • 对称性 即 DJS(p∥q)=DJS(q∥p)D_{JS}(p \| q)=D_{JS}(q \| p)D​JS​​(p∥q)=D​JS​​(q∥p), 而对称能让散度度量更准确。

参考: KL散度与JS散度 (opens new window)

# f Divergence

fff Divergence 长下面这个样子:(这是f Divergence的定义)

Df(P∥Q)=∫xq(x)f(p(x)q(x))dxD_{f}(P \| Q)=\int_{x} q(x) f\left(\frac{p(x)}{q(x)}\right) d x D​f​​(P∥Q)=∫​x​​q(x)f(​q(x)​​p(x)​​)dx

这里面要满足两个条件:

条件

  1. fff是凸函数,
  2. f(1)=0f(1)=0f(1)=0

fff取不同的函数，可以得到不同的Divergence:

例如, f(x)=xlogxf(x) = xlogxf(x)=xlogx, 得到：

Df(P∣∣Q)=∫xq(x)p(x)q(x)log(p(x)q(x))dx=∫xp(x)log(p(x)q(x))dxD_f(P||Q) = \int_{x} q(x) \frac{p(x)}{q(x)} \log \left(\frac{p(x)}{q(x)}\right) d x=\int_{x} p(x) \log \left(\frac{p(x)}{q(x)}\right) d x D​f​​(P∣∣Q)=∫​x​​q(x)​q(x)​​p(x)​​log(​q(x)​​p(x)​​)dx=∫​x​​p(x)log(​q(x)​​p(x)​​)dx

这其实就是KL Divergence;

又如, f(x)=−logxf(x) = -logxf(x)=−logx, 得到:

Df(P∥Q)=∫xq(x)(−log(p(x)q(x)))dx=∫xq(x)log(q(x)p(x))dxD_{f}(P \| Q)=\int_{x} q(x)\left(-\log \left(\frac{p(x)}{q(x)}\right)\right) d x=\int_{x} q(x) \log \left(\frac{q(x)}{p(x)}\right) d x D​f​​(P∥Q)=∫​x​​q(x)(−log(​q(x)​​p(x)​​))dx=∫​x​​q(x)log(​p(x)​​q(x)​​)dx

这其实就是Reverse KL Divergence;

再如, f(x)=(x−1)2f(x) = (x-1)^2f(x)=(x−1)​2​​, 得到:

Df(P∥Q)=∫xq(x)(p(x)q(x)−1)2dx=∫x(p(x)−q(x))2q(x)dxD_{f}(P \| Q)=\int_{x}q(x)\left(\frac{p(x)}{q(x)}-1\right)^{2} d x=\int_{x} \frac{(p(x)-q(x))^{2}}{q(x)} d x D​f​​(P∥Q)=∫​x​​q(x)(​q(x)​​p(x)​​−1)​2​​dx=∫​x​​​q(x)​​(p(x)−q(x))​2​​​​dx

这其实就是Chi Square Divergence.

也就是是说，f Divergence是各种Divergence的总称。
