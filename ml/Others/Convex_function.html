# 什么是上确界?

sup是supremum单词的缩写，意思是最小上界。inf是单词infimum的简写，意思是最大下界。 数学中，经常出现的表示方式是 lim sup 或者 lim inf，即找上界或者下界的极限。

sup{S}就是S的上确界，即S的所有上界组成的集合的最小元。

举例:

集合S=[0,1], sup{S} = 1, max{S} = 1;
集合S=[0,1), sup{S} = 1, max{S} 不存在。
